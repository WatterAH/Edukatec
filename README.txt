Instrucciones de uso:
> Crear una base de datos en MySQL con el nombre "users".
> En la consola de VSCode escribir el comando "npm start".
> Diríjase a su navegador predeterminado y escribir en la barra de navegación "localhost:8080"
> Ejecute la ruta de inicialización "/sam".
